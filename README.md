# 🎯 Minecraft Aim Assist Mod v2.0

Продвинутый мод для Minecraft 1.21.4, добавляющий настраиваемую систему помощи в прицеливании.

## ✨ Возможности

### 🎮 Основной функционал
- **Автоматическое наведение** на ближайшую цель в пределах FOV
- **Плавное прицеливание** с настраиваемой скоростью
- **Предсказание движения** цели для точных выстрелов
- **Прицеливание в голову** или центр масс
- **Проверка линии видимости** - не целится через стены
- **Гибкие настройки** через встроенное GUI меню

### ⚙️ Настройки

#### Основные параметры:
- **Скорость аима** (0.0 - 1.0) - насколько быстро прицел наводится на цель
- **Дистанция** (1-100 блоков) - максимальная дистанция для аима
- **FOV** (1-180°) - угол обзора для поиска целей
- **Плавное прицеливание** - включить/выключить smooth aiming
- **Предсказание движения** - учитывать скорость движения цели

#### Фильтры целей:
- ✅ Игроки
- ✅ Враждебные мобы
- ✅ Мирные мобы
- ✅ Животные
- ❌ Игнорировать союзников (teammates)
- ❌ Игнорировать невидимых

#### Дополнительно:
- Прицел в голову/центр масс
- Активация только с оружием в руках
- Вертикальная/горизонтальная компенсация
- Авто-прицеливание

## 🎹 Управление

| Клавиша | Действие |
|---------|----------|
| **R** | Включить/выключить Aim Assist |
| **Right Shift** | Открыть меню настроек |

## 📦 Установка

### Автоматическая установка (Рекомендуется)

1. Скачайте готовый `.jar` файл из [Releases](../../releases)
2. Поместите файл в папку `.minecraft/mods/`
3. Запустите Minecraft с Fabric Loader

### Сборка из исходников

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/minecraft-aimassist.git
cd minecraft-aimassist

# Соберите мод
./gradlew build

# Готовый файл будет в build/libs/
```

## 🛠️ Требования

- **Minecraft**: 1.21.4
- **Fabric Loader**: 0.16.10+
- **Fabric API**: 0.102.0+
- **Java**: 21+

## 📝 Конфигурация

Конфигурационный файл создается автоматически при первом запуске:
```
.minecraft/config/aimassist.json
```

Пример конфига:
```json
{
  "enabled": true,
  "aimSpeed": 0.5,
  "aimDistance": 16.0,
  "aimFOV": 45.0,
  "smoothAiming": true,
  "predictMovement": true,
  "targetPlayers": false,
  "targetHostile": true,
  "aimAtHead": true
}
```

## 🔧 Для разработчиков

### Структура проекта
```
src/main/java/com/aimassist/
├── AimAssistMod.java          # Главный класс мода
├── config/
│   ├── AimAssistConfig.java   # Система конфигурации
│   └── ConfigScreen.java      # GUI меню настроек
└── util/
    └── AimAssistHelper.java   # Логика прицеливания
```

### API классы

#### AimAssistConfig
Управление настройками мода:
```java
AimAssistConfig config = AimAssistMod.getConfig();
config.setAimSpeed(0.7);
config.setTargetPlayers(true);
config.save();
```

#### AimAssistHelper
Основная логика прицеливания:
```java
// Применить аим-ассист
AimAssistHelper.applyAimAssist(client, config);

// Найти лучшую цель
LivingEntity target = AimAssistHelper.findBestTarget(client, config);
```

## 🐛 Известные проблемы

- В некоторых случаях может конфликтовать с другими модами, изменяющими камеру
- Предсказание движения может быть неточным для быстро движущихся целей
